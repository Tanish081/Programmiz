# Programmiz

Programmiz is a Flutter learning app built around a Duolingo-style flow for beginner-friendly Python practice.

## What it includes

- Splash, placement quiz, onboarding, and language hub flow
- Python lesson path with quizzes, XP, streaks, and local progress saving
- Duolingo-inspired UI components and green-first theme
- Audio feedback and mobile-friendly layouts

## Requirements

- Flutter SDK
- Android Studio or VS Code with Flutter support
- A connected device or emulator

## Run locally

```bash
flutter pub get
flutter run
```

## Project structure

- `lib/` - app source code
- `assets/` - audio, curriculum, and logo assets
- `android/`, `ios/`, `web/`, `windows/`, `macos/`, `linux/` - platform targets

## Notes

- The app currently focuses on Python lessons.
- Progress, onboarding, and placement results are stored locally on the device.
